# CSDB - C64 Scene Database

You can browse [csdb.dk](https://www.csdb.dk/) and retrieve demos and games from there. You can download or run them.

Requirement: C64 Ultimate or 1541 Ultimate cartridge with networking capabilities.

## Before You Begin

To activate CSDB commands, first switch to virtual device C by typing:

```
#C
```

## Navigating CSDB

### search <term>

You can search releases, groups and sceners. Next select from the list by typing its number to get to details.

## Show Detail by id

- `rls 12345` - Get release info
- `grp 678` - Get group info
- `scn 999` - Get scener info
