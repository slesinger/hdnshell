# Filesystem Operations

## Ultimate Storage

`mkdir <name>` — Create a new directory with the specified name.

## ??

- `[<device letter>]save "<name>" [<from address>] [<to address>]` — Save file to disk or tape.
- `[<device letter>]delete "<name>"` — Delete file from disk.

save "backup"      # Save to default device
9:delete "old.prg" # Delete 'old.prg' from device 9
